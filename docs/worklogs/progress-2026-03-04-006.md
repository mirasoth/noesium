# Common config for builtin subagents (headless/headed)

## Objective
Implement the plan: add optional `config` object on builtin subagent entries, support `headless` for browser_use, and flow config from example_config.json through to BrowserUseAgent.

## Completed
- Added `config: Optional[Dict[str, Any]] = None` to core `AgentSubagentConfig`
- Added `config` to noeagent `AgentSubagentConfig` and passed it in `from_global_config()`
- Wired builtin factory via closure over subagent_cfg; `_create_browser_use_agent(subagent_cfg)` reads `config.headless` (default True) and passes to BrowserUseAgent
- Updated `_create_tacitus_agent` to accept optional `subagent_cfg` for API consistency
- Added `"config": { "headless": false }` to browser_use in example_config.json
- Documented `config` and browser_use `headless` in docs/impl/noeagent-config-system.md
- Added unit tests for config parsing and factory headless behavior

## Files Changed
- `noesium/core/config.py` – AgentSubagentConfig + config field
- `noesium/noeagent/config.py` – AgentSubagentConfig + config; from_global_config passes config and enabled
- `noesium/noeagent/agent.py` – closure-based factory binding; _create_browser_use_agent(cfg) with headless; _create_tacitus_agent(cfg) signature
- `noesium/noeagent/example_config.json` – browser_use entry + config.headless: false

## Notes

- Key finding: `SUBAGENT_PROGRESS` and `SUBAGENT_END` event types exist in the enum but are never emitted. `interact_with_subagent()` calls `child.arun()` which swallows all intermediate progress.
- Key finding: `initialize()` runs on every `astream_progress()` call, re-creating registry, executor, memory, and graph â€” estimated 200-500ms overhead per query after the first.
- Proposed callback-to-yield bridge pattern (asyncio.Queue) to forward subagent events through `astream_progress()` generator.
- Highest-priority optimization: O1 (init guard) fixes B1/B2/B3/B6 with minimal code change.

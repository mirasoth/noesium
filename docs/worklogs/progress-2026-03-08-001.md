## Key Design Decisions

1. **3-field model**: Kept minimal with goal, findings, scratchpad
2. **FIFO auto-trim**: Prevents unbounded memory growth
3. **Subagent isolation**: Child context doesn't inherit parent state
4. **Optional memory**: Persistence is opt-in via save/load methods
5. **Event-sourcing removal**: Simplified architecture without EventStore/EventEnvelope

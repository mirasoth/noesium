## Objective

Clean up RFC specifications to remove out-of-date design elements (AgentRef, TraceContext, SignatureBlock, EventEnvelope, EventStore, DomainEvent) that have been deprecated and removed from the implementation.

## Completed

- [x] RFC-1001: Added deprecation notices to AgentRef (§5), TraceContext (§6), SignatureBlock (§11) sections
- [x] RFC-1001: Updated Section 19 to document removed components and current implementation alternatives
- [x] RFC-2001: Added implementation status noting deprecated components
- [x] RFC-2001: Updated package structure to reflect actual state (removed kernel/, projection/, durable.py, bridge.py)
- [x] RFC-2001: Added deprecation notices to Event System, Projection Layer, DurableMemory, BaseAgent sections
- [x] RFC-2003: Removed EventStore and AgentRef dependencies from CapabilityRegistry
- [x] RFC-2003: Updated to Provider-First design without hybrid event sourcing
- [x] RFC-2102: Added deprecation notice for EventSourcedProvider
- [x] RFC-2103: Simplified ToolExecutor to operate without event emission
- [x] RFC-2103: Updated ToolContext to remove TraceContext
- [x] rfc-namings.md: Removed 21 deprecated terms, added ProgressEvent
- [x] rfc-history.md: Added 2026-03-09 entries documenting spec cleanup

## Notes

**Deprecated Terms Removed from rfc-namings.md:**
- AgentRef, Causation ID, Cognitive Projection, Correlation ID
- Domain Event, Durable Memory, Envelope Bridge
- Event Envelope, Event Store, EventSourcedProvider
- Execution Projection, Federated Projection
- Kernel Executor, Memory Hierarchy, Node Result, Partition Key
- Projection, Projection Engine, Projection Layer
- Signature Block, Trace Context

**Current Implementation Alternatives:**
- EventEnvelope → ProgressEvent (TUI streaming only)
- AgentRef → Simple dict or omitted
- TraceContext → LangGraph callbacks, OpenTelemetry
- EventStore → Local changelog in CapabilityRegistry
- DurableMemory → Memu file-based persistence
- Projection → Direct state management, CognitiveContext

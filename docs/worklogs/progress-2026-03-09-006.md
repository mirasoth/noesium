## Notes

**Content Completely Removed:**
- EventEnvelope and all related structures (AgentRef, TraceContext, SignatureBlock)
- DomainEvent base class and event type definitions
- EventStore interface and implementations
- ProjectionEngine and BaseProjection
- KernelExecutor with event emission
- EventSourcedProvider and DurableMemory
- EnvelopeBridge

**Current Implementation (retained in specs):**
- ProgressEvent for TUI streaming
- CognitiveContext (RFC-1010) for state continuity
- MemuProvider for persistent memory
- WorkingMemoryProvider for session memory
- Local changelog in CapabilityRegistry
- LangGraph checkpointer for persistence
- OpenTelemetry/Opik for tracing

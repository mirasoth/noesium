## Objective

Implement the Core Subagents Design Proposal to align the codebase with the specification for PlanAgent and ExploreAgent.

## Completed

### PlanAgent Implementation
- Created `schemas.py` with Pydantic models: ActionDetail, VerificationStep, PlanStep, Dependency, DetailedPlan, ContextEvaluation, Requirement, Constraint
- Updated `state.py` with unified context handling fields
- Replaced `prompts.py` with general-purpose domain-agnostic prompts
- Updated `agent.py` with new workflow: evaluate_context -> explore_resources (conditional) -> analyze_requirements -> generate_plan -> define_verification -> finalize_plan
- Added plan type detection and structured output using Pydantic models

### ExploreAgent Implementation  
- Created `schemas.py` with Pydantic models: Finding, Source, ReflectionResult, ExploreResult, SearchQuery, TargetAnalysis, SearchStrategy

- Replaced `prompts.py` with general-purpose domain-agnostic prompts
- Updated `agent.py` with reflection loop: analyze_target -> generate_strategy -> explore_sources -> reflect -> [loop if insufficient] -> synthesize
- Added confidence score calculation in reflection and synthesis nodes

### Module Updates
- Updated `plan/__init__.py` to export new schemas
- Updated `explore/__init__.py` to export new schemas
- Updated proposal status to "Implemented" and marked migration checklist complete

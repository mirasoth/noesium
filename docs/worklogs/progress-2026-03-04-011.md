## Update: Fixed Dead Loop Issue + Renamed wizsearch to web_search

### Dead Loop Fix

**Problem:** LLM kept trying to invoke Tacitus with `invoke_builtin` action, getting rejected, and retrying with different message prefixes.

**Root Cause:** Error message was confusing - told LLM to use `/tacitus` prefix in message, but the action was still `invoke_builtin`.

**Solution:** Updated error message in `nodes.py`:
```python
result = (
    f"{display_name} cannot be auto-invoked. "
    f"It requires explicit user command (/research or /deep_research in the original message). "
    f"For this task, use web_search tool instead to search for information. "
    f"Do NOT try to invoke {display_name} again."
)
```
